# Asian Tech Influencers Analysis | Week 1 Internship Task

This repository contains my **Week 1 Task** for the Influencer Marketing Internship.

**Objective**: Collect, clean, and analyze a dataset of 50–100 tech-focused influencers from Asian countries across multiple platforms (YouTube, TikTok, Instagram/Meta, X, and Google).

### 📋 Task Summary
- Collected tech influencers from **China, South Korea, Japan, India, Singapore, UAE, Thailand, and Pakistan**
- Platforms covered: **YouTube, TikTok, Meta (Instagram), X (Twitter), Google**
- Performed data cleaning, merging, and feature engineering
- Created visualizations and business insights on influencer marketing


### 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib & Seaborn
- Jupyter Notebook

### 📊 Visualizations Included
- Engagement Rate Distribution
- Virality Score Distribution
- Tech vs Non-Tech Engagement Comparison
- Top 10 Brands by Engagement


Full detailed report is also available in the folder.
